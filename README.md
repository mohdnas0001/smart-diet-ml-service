# Smart Diet ML Service

AI-powered Food Recognition & Nutrient Estimation microservice for the Smart Diet and Food Analyzer system.

## Tech Stack
- Python 3.11 + FastAPI
- PyTorch + YOLOv8 + EfficientNet-B4
- USDA FoodData Central & Nutritionix API integration
- Docker + Azure Cloud deployment

## Module
**Member 1**: Food Recognition & Nutrient Estimation Module

## System Integration
This microservice communicates with the ASP.NET Core (C#) backend via internal HTTP API calls.
